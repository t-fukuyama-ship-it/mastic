# Mastic Research Page

This folder is a GitHub Pages-ready static website for the bilingual Mastic research introduction page.

## Files

- `index.html`: Main bilingual Japanese/English page.
- `assets/`: Optimized images used by the page.
- `.nojekyll`: Keeps GitHub Pages from processing the site with Jekyll.

## GitHub Pages Use

1. Upload all files in this folder to the root of a GitHub repository.
2. In the repository settings, enable GitHub Pages.
3. Select the branch and root folder that contain `index.html`.

## Notes

- The page uses local image paths under `assets/`.
- The language switch is built into `index.html`.
- Scientific names are rendered with a dedicated `.taxon` class.
- Reference entries include PubMed links.
- Chios Island visit photos are included in the horizontal photo card section.
